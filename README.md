# 📚 Student Task Manager - MAD Lab Project

A modern and responsive **React Native Expo mobile application** designed to help students manage their academic tasks efficiently.

The application allows students to organize assignments, quizzes, labs, and projects with task tracking, file submission, notifications, and a professional mobile UI.

---

# 📱 Project Overview

Student Task Manager provides an easy way for students to manage their academic workload.

Students can create tasks, set deadlines, track progress, submit completed work files, and manage their productivity through a clean dashboard.

This project demonstrates important React Native concepts including:

- Component Based Architecture
- React Navigation
- Redux State Management
- AsyncStorage Data Persistence
- File Handling
- Theme Management

# 🚀 Installation & Setup

Clone repository:

```bash
git clone repository-link
```

Go inside project:

```bash
cd StudentTaskManager
```


Install dependencies:

```bash
npm install
```


Start Expo server:

```bash
npx expo start
```


Run on:

- Android Emulator
- Expo Go Mobile App
- Web Browser
